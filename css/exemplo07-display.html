Displays

A propriedade display no CSS é utilizada para definir como os elementos são renderizados na página, ou seja, como eles se comportam visualmente e interagem com os outros elementos. Essa propriedade é essencial para estruturar layouts e ajustar a disposição de elementos na interface. Para a propriedade display temos os seguintes valores disponíveis: block, inline, inline-block, flex (veremos em um exemplo separado.)

Display: block

Ocupam toda a largura disponível e iniciam uma linha automaticamente. Tags que são display:block p, h1, div

Display: inline

Ocupam apenas o espaço necessário para o conteúdo e não iniciam uma nova linha. Tags que são display:inline img, button, a, span, input.

Display: inline-block

São semelhantes a elementos inline, mas permitem definir largura e altura dos elementos. Eles não iniciam uma nova linha. Quando posso utilizar? Quando vc tem um elemento que deseja que tenha o comportamento inline(ocupe o menor espaço possível), porém definindo o width e/ou height.
