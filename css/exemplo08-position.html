Position

A propriedade position determina como um elemento é posicionado dentro de seu contâiner ou na tela.

Static

static é o valor padrão. O elemento é posicionado no fluxo normal do documento, sem ser afetado por outras propriedades de posição

Static

Relative

relative posiciona o elemento em relação à sua posição original no fluxo do documento. O elemento se move, mas o espaço original ainda é ocupado. Utilizado as propriedades: top, right, bottom e left para mover o elemento à sua posição original.

Relative

Fixed

fixed posiciona o elemento em relação à janela de visualização (viewport) (navegador), ou seja, ele permanece fixo na tela ao rolar a página.

Fixed

Absolute

absolute posiciona o elemento em relação ao seu container pai mais próximo que tenha um position diferente de static. Se não houver tal contâiner, serã posicionado em relação ao elemento raiz (geralmente o body).

Absolute

Sticky

sticky é uma combinação de relative e fixed. O elemento se comporta como um elemento relativo até atingir um determinado ponto da tela, onde se fixa.

Sticky
