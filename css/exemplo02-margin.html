Margin

O que são margins?

A propriedade margin no CSS é usada para criar espaçamentos externo ao redor de um elemento. Ela afasta o elemento de outros ao seu redor.

Propriedades específicas:

  • margin-top: Define o espaço acima do elemento.
  • margin-right: Define o espaço à direita do elemento.
  • margin-bottom: Define o espaço abaixo do elemento.
  • margin-left: Define o espaço à esquerda elemento.
