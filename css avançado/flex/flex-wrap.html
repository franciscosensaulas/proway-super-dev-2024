Exemplos de flex-wrap

Define se os itens devem "quebrar" para uma nova linha/coluna quando não houver espaço disponível no eixo principal.

flex-wrap: nowrap

Descrição: Os itens não quebram para a próxima linha. Se a soma das larguras for maior que a do container, surgirá uma barra de rolagem horizontal.
