Exemplos de justify-content

O que é justify-content? É a propriedade que determina como itens em um container flex são distruibuídos ao longo do eixo principal. Se o eixo principal for horizontal (padrão para flex-direction: row), os itens são alinhados horizontalmente de diferentes formas, dependendo do valor escolhido.

justify-content: flex-start

Descrição: Os itens são colocados no início do container (lado esquerdo, no caso de direção em linha).
