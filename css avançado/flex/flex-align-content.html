Exemplos de align-content

O que é align-content? É a propriedade que determina como as linhas (ou colunas) dentro de um contêiner flex são distribuídas ao longo do eixo transversal. Para que funcione, é preciso haver múltiplas linhas — ou seja, flex-wrap precisa estar ativado, e os itens devem ocupar mais de uma linha.

align-content: flex-start

Descrição: As linhas são agrupadas no início do eixo transversal (parte superior, se for flex-direction: row).
