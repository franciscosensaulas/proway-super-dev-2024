CSS Transform e Rotate

A propriedade transform no CSS permite aplicar transformações a um elemento, como redimencionamento, rotação, inclinação ou movimentação. É uma ferramenta poderosa, que permite criar efeitos visuais sem a necessidade de scripts.

O que é rotate?

A função rotate dentro da propriedade transform é usada para girar um elemento em torno de um ponto de origem. A rotação é especificada em graus deg, mas também pode usar outras unidades como radianos rad ou frações de volta turn.

Exemplos básicos:

transform: rotate(45deg) - Este código gira um elemento em 45 graus no sentido horário.
