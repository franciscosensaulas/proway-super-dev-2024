CSS Transform Skew

O que é skew?

A função skew no CSS é usada para inclinar ou distorcer um elemento ao longo de seus eixos X ou Y. A inclinação é especificada em graus deg.

Tipos de Skew:

  • skewX(valor): Inclina o elemento ao longo do eixo X.
  • skewY(valor): Inclina o elemento ao longo do eixo Y.
  • skew(x, y): Inclina o elemento ao longo dos eixos X e Y.
