O que é translate?

A função translate no CSS é usada para mover um elemento no espaço bidimensional (horizontalmente e verticalmente). É especificada em unidades como px, %, ou em.

Tipos de translate:

  • translateX(valor): Move o elemento ao longo do eixo X.
  • translateY(valor): Move o elemento ao longo do eixo Y.
  • translate(x, y): Move o elemento ao longo dos eixos X e Y.

Exemplos:

translate X 50px
translate Y 50px
translate X Y 50px
translate active
